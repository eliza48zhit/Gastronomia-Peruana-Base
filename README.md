🇵🇪 Gastronomía Peruana - Smart Contract (Base Mainnet)
Este es un contrato inteligente dedicado a la milenaria y variada cultura culinaria de Perú, desplegado en la red Base. Un proyecto recreativo y educativo que utiliza la tecnología blockchain para registrar platos icónicos y reconocidos mundialmente como el Ceviche, el Lomo Saltado y el Ají de Gallina.

🔗 Verificación en Basescan

El contrato ha sido verificado satisfactoriamente para permitir la lectura y escritura pública del código. Esto permite que cualquier "chef" digital pueda actualizar el menú directamente en la blockchain, asegurando que el sabor peruano quede registrado de forma inmutable.

📖 Funcionalidad

Este contrato ofrece una interfaz simplificada para la gestión de datos on-chain:

Lectura del Menú: A través de la función queHayDeComer, cualquier usuario puede consultar el plato y los ingredientes actuales almacenados en el contrato sin costo de gas.

Actualización del Menú: Mediante la función actualizarMenu, los usuarios pueden proponer un nuevo plato típico y su descripción, generando una transacción real en la red Base y actualizando el estado global del contrato.
